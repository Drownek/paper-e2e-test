# Wiki Pages

This directory contains the GitHub Wiki pages for the Paper E2E Test Framework.

## Publishing to GitHub Wiki

To publish these pages to your GitHub wiki:

1. Clone the wiki repository:
   ```bash
   git clone https://github.com/yourusername/paper-e2e-test-framework.wiki.git
   ```

2. Copy all `.md` files from this directory to the wiki repository

3. Commit and push:
   ```bash
   git add .
   git commit -m "Update wiki documentation"
   git push
   ```

## Wiki Structure

- `Home.md` - Wiki home page with navigation
- `Getting-Started.md` - Installation and setup guide
- `Writing-Tests.md` - Test writing guide and patterns
- `Matchers-Reference.md` - Complete assertion API reference
- `GUI-Testing.md` - GUI testing guide
- `TypeScript-Support.md` - TypeScript configuration
- `Configuration.md` - Gradle plugin options
- `Troubleshooting.md` - Common issues and solutions
- `Examples.md` - Real-world test examples

## Editing

Edit the markdown files in this directory and republish to update the wiki.
